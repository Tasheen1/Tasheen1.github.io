Girls Who Code

About

Girls Who Code is a program fueled by Concord Public Library, where young girls take on obstacles of coding to defy gender barriers, and help the face of computer science. Each year we make life changing projects, to help the environment, and the people we live with. We all work together, as a club, and a group, to make these websites, that we are proud of. Oh, and We Code Like Girls, So Try To Keep Up!

Sophia Hubscher

Facilitator

Dorrie Karlin

Facilitator & Branch Librarian

Dorrie Karlin is the Fowler branch librarian for the Concord Free Public Library, and she feels so lucky to get to see the Concord Girls Who Code in action. This year, she has watched them learn new languages, get to know each other, and figure out how to work together all online. She is a connection to the library and a cheerleader, and she's so proud of the work the club has done.
